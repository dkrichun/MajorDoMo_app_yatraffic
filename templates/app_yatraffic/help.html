Вызов модуля в меню

[#module name="app_yatraffic"#] - вывод сведений о пробках

Вызов модуля в своих скриптах/методах

Для "ручного" обновления данных, в своих скриптах/методах можно использовать:
include_once(DIR_MODULES.'app_yatraffic/app_yatraffic.class.php');
$yatraffic=new app_yatraffic();
$yatraffic->get_traffic(gg('yt_setting.reg_id'));
Для автоматического обновления нужно вставить в метод OnNewHour (или любой другой, выполняемый ежечасно) следующий код:
$updateTime = gg('yt_settings.updateTime');
if($updateTime > 0){
$count = gg('yt_settings.countTime');
if($count >= $updateTime){
include_once(DIR_MODULES.'app_yatraffic/app_yatraffic.class.php');
$yatraffic=new app_yatraffic();
$yatraffic->get_traffic(gg('yt_settings.city_id'));
sg('yt_settings.countTime',1);
} else {
$count++;
sg('yt_settings.countTime',$count);
}
}
